## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
4. Mobile User Guide
Experience the Full Interface: > For the best experience on mobile devices, please click the "Full Display/Preview" button located at the top-right corner next to the logo. This ensures you can access all the advanced features of AURYX AI without any UI clipping.
