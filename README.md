🛠️ Developer Setup (Local
 Deployment)
Prerequisites
Node.js (v18+)
Step-by-Step Installation

Step 1: Clone the repository
Clone the repository to your local machine and navigate into the project folder using the terminal.

Step 2: Install dependencies
Install all necessary packages to run the project by running the npm install command in your terminal.

Step 3: Configure Environment
Create a .env.local file in the project's root directory. Open the file and add your Gemini API key in the format GEMINI_API_KEY=your_api_key_here. Note: Never share your .env.local file publicly.

Step 4: Launch the application
Start the application in your local environment by running the npm run dev command.
